# 작성 컨벤션 (엄격 준수)

## 변수 명명
- 영문 **UPPER_SNAKE_CASE**.
- 시점(매월 변하는) 변수는 끝에 `_T` (예: `AV_END_T`, `INFORCE_T`).
- 집계/스칼라(시점 무관)는 `_T` 없이 (예: `BEL_AT_ISSUE`, `CSM_AT_ISSUE`).

## 타입 분류
- `Input (입력)` — 모델포인트 입력값 (성별·가입나이·가입금액·기본보험료 등)
- `Parameter (상수)` — 계약 단위 상수 (납입기간·투영종료나이·요율 가정 등)
- `MS (시점)` — 매월 계산되는 시점 변수
- `Boolean` — 0/1 플래그 (단계·면책경과·납입진행 등)
- `Status` — 차원 키(성별 등) 또는 단계 상태
- `TableRef` — 테이블 참조 변수
- `Numeric (집계)` — 현가합·초기인식값 등 시점 무관 집계

## 포뮬라 컨벤션
- 함수: `PREVIOUS(var)` (전월값), `TABLE("id", key1, key2)` (테이블 조회), `SUM(expr)` (전기간 합), `SUM_FROM_T(expr)` (t시점~끝 합), `IF(cond, a, b)`, `MAX/MIN`, `FLOOR`, `MOD`.
- 거듭제곱은 `^`. `t` = 월 인덱스(0-based).
- **연→월 환산은 정확식** `1 - (1 - q)^(1/12)` (근사 `q/12` 금지).
- 월 복리 수익률 `(1 + r_annual)^(1/12) - 1`.
- 가정 상수는 샘플 숫자 + 주석: `0.035 // 샘플 — 회사 투자가정으로 교체`.
- 근사식은 의미란에 "근사" 명시 + 필요시 `// 근사식` 주석.

## 표준 영역 골격
입력 → 상수 → 시간 인덱스 → 테이블 바인딩 → 감소율(유지·사망·해약) → 급부 지급 → (변액/저축성이면) 특별계정·펀드 → 보험료 → 사업비 → 현금흐름 → 준비금 → 할인 → K-IFRS17(BEL·RA·CSM). 영역 8~11개, 각 영역 변수 4~9개.

## 상품유형별 적응
- **암·건강**: 담보별(진단·사망·소액암·수술·입원) 월 기대지급 = 면책경과 × 발생률(월환산) × 가입금액 × 평균유지율. 입원일당은 확률이 아니라 "연 기대일수/12". 갱신형이면 최초계약만 투사(갱신 = 새 GoC). 면책 플래그를 모든 암담보에 공통 곱.
- **변액유니버설·종신**: 계약자적립액 롤포워드가 모델의 심장 — `AV_END_T = MAX(0, (AV_START_T + PREM_INVESTED_T - MONTHLY_DEDUCTION_T) * (1 + FUND_RETURN_M_T - FUND_FEE_M_T))`. 사망보험금은 보통 `MAX(가입금액, 적립액×배수, 기납입보험료)` 3중 구조. 특정 회차(예: 24회) 경과 시 비용구조 전환 플래그. NAR=순보장위험액, COI=NAR×월사망률.
- **변액연금**: `PHASE_T = IF(t < DEFER_MONTHS, 1, 2)` (적립기/연금기)를 최상단 분기로. 연금기준금액 롤포워드, 최저보증연금(실적연동 100~120% 초과지급), 연금기 사망보증, 보증 초과지급(회사 부담). 적립기에만 해약 발생.
- **최저보증(GMxB)**: 보증 정의 → 보증비용(월공제) → 보증 초과지급(펀드 소진 후 회사 부담)을 분리.

## K-IFRS17
- 현금흐름 현가합: `PV_x = SUM(x_T * DISCOUNT_FACTOR_T)`.
- `BEL = PV(유출) - PV(유입)` (결정론 근사 시 "TVOG 미포함" 명시).
- `RA` = 충격 기반 근사(위험률·해약률 ±, 연금은 장수=사망률 하향 포함), 신뢰수준 75% 가정 명시.
- `CSM_AT_ISSUE = MAX(0, -(BEL + RA))`, 음수면 손실부담계약→손실요소.
- CSM 롤포워드: 기시잔액 → (일반계정 GMM이면 locked-in 이자 accretion / 변액·연금 VFA이면 기초항목 변동 중 회사몫 조정) → Coverage Unit 비율로 상각(release) → 기말잔액.
- Coverage Unit: 보장서비스(NAR)와 투자서비스(적립액)를 상품에 맞게 합산. 변액연금은 적립기=적립액, 연금기=보증연액 근사.
